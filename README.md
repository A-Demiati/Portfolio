# Ahmad Demiati Portfolio

Personal portfolio website showcasing skills, projects, and experience.

## 🚀 Quick Start

### Running Locally

1. **Make sure you have Node.js installed**
   - Download from: https://nodejs.org/

2. **Navigate to the project folder**
   ```bash
   cd path/to/your/portfolio
   ```

3. **Start the server**
   ```bash
   node server.js
   ```

4. **Open your browser**
   - Go to: http://localhost:5401

5. **Stop the server**
   - Press `Ctrl+C` in the terminal

## 📝 Customization

### Update Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --accent-color: #7b2cbf;  /* Main purple */
    --bg-color: #0d0d0d;      /* Background */
    /* ... more variables */
}
```

### Update Content
- **Personal Info**: Edit the HTML files directly
- **Projects**: Modify the project cards in `portfolio.html`
- **Skills**: Update percentages in `about.html`
- **Social Links**: Already configured with your profiles

## 🌐 Deployment

### Option 1: GitHub Pages
1. Create a GitHub repository
2. Push your files
3. Enable GitHub Pages in Settings

### Option 2: Netlify/Vercel
1. Drag and drop your folder
2. Instant deployment

### Option 3: Traditional Hosting
1. Upload all files to your web host
2. Point domain to the directory
